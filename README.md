# Purple Team / Red-Purple Hybrid Lab

## Overview
This lab demonstrates combined offensive and defensive skills: simulating phishing attacks and assessing endpoint vulnerabilities.

## Objectives
- Simulate lab-based attack vectors
- Assess endpoint vulnerabilities
- Suggest remediation & mitigation (Purple Team mindset)

## Tools
- PowerShell, OSQuery
- Nmap / OpenVAS
- Python automation scripts

## Structure
- `Lab-Setup.md` : Instructions to recreate lab
- `Reports/` : Findings & mitigation reports
- `Scripts/` : Automation scripts
- `Screenshots/` : Lab evidence

## Outcome
Demonstrates Red/Purple Team mindset, VA skills, and attack/defense workflow understanding.
